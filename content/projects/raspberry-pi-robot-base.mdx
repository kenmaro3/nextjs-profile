## Project overview

ESP32 より一段上の処理量が必要になったときのために、Raspberry Pi ベースの移動体構成をまとめたプロジェクトです。

## Purpose

映像処理や高レベル制御の余地を持ちながら、どこまでコンパクトにまとめられるかを確認することが狙いです。

## Circuit / CAD / 3D print notes

この構成では、基板固定位置とケーブル長が熱とメンテナンス性に直結するので、ケースより先に内部レイアウトを固めました。

## Build log

基本的な移動体としては成立したので、今後は用途を明確にした上で派生構成へ分ける方が良さそうです。

## Failures so far

<Callout title="Constraint that stayed visible" tone="warning">
  Raspberry Pi を載せると自由度は増えますが、消費電力と発熱の管理が一気に難しくなります。
</Callout>
