## Project overview

机上で扱えるサイズ感の小型4輪ロボットを作り、移動、電源、センサー追加、シャーシ改善を段階的に試していくプロジェクトです。

## Purpose

目的は「最初からすべてを載せた完成機」を作ることではなく、ハードウェア試作の土台を持つことです。

- モーター制御の基礎を詰める
- ESP32 を中心とした制御構成を固める
- 3Dプリント部品の改善サイクルを回す
- 失敗した構成もそのまま残す

## Circuit / CAD / 3D print notes

現時点では、回路は最短で走行確認できる構成に寄せています。CAD と 3Dプリントは、フレーム剛性よりも「すぐ刷り直せること」を優先しています。

## Build log

最初の段階では、まずシャーシ寸法とモーター固定位置を決め、ESP32 から基本的な動作確認をしています。

## Failures so far
